# candylovable

A basic project scaffold.

## Getting started

1. Copy the example environment file and fill in your own values:

   ```bash
   cp .env.example .env
   ```

2. Add your secrets to `.env` (this file is git-ignored and never committed).

## Environment variables

See [`.env.example`](./.env.example) for the variables this project expects.
